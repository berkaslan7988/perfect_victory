# Disclaimer & Responsible Use

This repository is a **personal educational portfolio**. Several projects implement
offensive-security techniques — including a keylogger, a backdoor simulator, a
network packet sniffer, password/PIN brute-forcers, a web directory brute-forcer,
and a MAC-address spoofer simulator.

These were built **solely to understand how such techniques work**, so that they
can be detected and defended against. They are deliberately simple, well-commented
learning exercises.

## Rules of use

- ✅ Use these tools **only** on systems, networks, and accounts that **you own**
  or for which you have **explicit, written permission** to test.
- ❌ Do **not** use any code here against systems you do not own or are not
  authorized to assess.
- ⚖️ Unauthorized access to computer systems is illegal in most jurisdictions
  (e.g. the Computer Fraud and Abuse Act, the UK Computer Misuse Act, and similar
  laws worldwide).
