# Professional Network Packet Sniffer

Captures and parses live network traffic with `scapy`, decoding protocol layers
(Ethernet/IP/TCP/UDP) — a real look at what flows across a network.

```bash
pip install scapy
sudo python professional_network_packet_sniffer.py
```

> Capture only on networks you own or are authorized to monitor. See ../../DISCLAIMER.md.
